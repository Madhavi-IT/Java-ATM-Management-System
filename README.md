# Java-ATM-Management-System
A console-based ATM Management System built using Core Java, JDBC, and MySQL.
## Technologies
- Java
- JDBC
- MySQL
- OOP Concepts

## Features
- Login using card number and PIN
- Balance enquiry
- Deposit and withdraw money
- Database connectivity

## How to Run
1. Clone the repository
2. Create database using atm.sql
3. Update DB credentials
4. Run Main.java
